# Irrigate!!!

## What is Irrigate?
Irrigate is a Minecraft mod aimed at allowing farmers, or really anyone who wants to plant crops, to see whether their farmland is irrigated or not.

## How do we determine if a block is irrigated or not?
A single water block can hydrate a 9X9 square of farmland. A lot of people already know this, but a lot of people also don't want to remember that rule every time they're farming, especially if their farm is huge and they don't want to keep track of all of that.

## So how does this mod actually work?
When the (default) key "I" is pressed, the mod scans the area around you for farmland, and if it finds any then scans a 9X2X9 cube around that farmland to check for water, which is exactly what the game itself does to determine whether a block of farmland can be hydrated or not. If it can be, a blue overlay is rendered over the top of the block. If it can't, a red overlay will be rendered the same way. This can be toggled on and off using "I" by default.

## Wait, but... doesn't this already exist?
Kind of. There have been other mods in the past that have done similar, but they are either outdated, no longer maintained or skip versions. The goal of this mod is to bring that functionality to the newer versions, without skipping a single one. That's why this mod is available for all 1.21.x and 26.x.x versions.
